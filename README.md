## 🛠️ Requirements

- Python 3.10+
- Flask

Install dependencies:

```bash
pip install flask
```

---

## ▶️ Usage

Run email test:

```bash
python mail_test.py
```

Run QR script:

```bash
python qr_test.py
```

---

## ⚙️ Configuration

Edit credentials in `mail_test.py`:

```python
EMAIL = "your_email@gmail.com"
PASSWORD = "your_app_password"
```
